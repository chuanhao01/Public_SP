SP Quiz Application This was a quiz application I made using bootstrap-vue for my school programming assignment.
I used this framework as it allowed me to focus more on coding the logic behind the quiz rather than messing with the front-end html and js.
As vue already did the 2-way binding of elements and variables for me, while bootstrap allowed me to use componenets, all I had to do was implement the logic for the quiz. 		I was limited in the data structure I had to use for both my quiz and mcq classes.
The school assignemnt specified that I had to store the questions as an array and use its index as a unique ID for the question.
Inspite of this limitation, I managed to implement a shuffle function for both the questiuons and options.
Pomodoro timer This was a small project I tried doing in python as I wanted my own customisable pomodoro timer set-up.
I had difficulty in the past as I did not know how to handle asynchronous calls in python for the timer. I also had a hard time dealing with time in python.
Now, I managed to solve this problem using in JS using polling instead of interupt. 		In the future I hope to implement this timer in async and use interput instead, in order to make the timer more efficent.
